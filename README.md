# Natalia Rico — Personal Portfolio (oxlusio.github.io)

This repository holds the source for my personal portfolio site published at:
https://oxlusio.github.io

Portfolio of Natalia Rico — Computer Engineering student at the University of New Mexico. The site showcases academic and technical projects including FPGA work, data dashboards, and web demos.

Contact
- Email: nrico@unm.edu

Contents
- index.html — main markup with sections: Home, About, Projects, Skills, Contact
- css/styles.css — styles (mobile-first)
- js/script.js — client-side behavior (nav, project filter, contact form)
- assets/ — add images, resume PDF (place resume at assets/Natalia-Rico-Resume.pdf)

Quick start (publish to GitHub Pages)
1. Make sure this repo is named `oxlusio.github.io` (published at https://oxlusio.github.io).
2. Put `index.html` at the repository root and commit the `css/` and `js/` folders.
3. Push to your default branch (main or master). Example commands:

   git init
   git add .
   git commit -m "Initial site — Natalia Rico portfolio"
   git branch -M main
   git remote add origin https://github.com/oxlusio/oxlusio.github.io.git
   git push -u origin main

4. Visit https://oxlusio.github.io (allow a minute for Pages to publish). Check Settings → Pages for status.
